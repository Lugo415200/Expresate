# Food Flashcard Images

Place optimized local food photos in this folder.

Recommended format:
- JPG or WebP
- About 600x400 pixels
- Consistent 3:2 aspect ratio
- Compressed for web use
- Licensed from Unsplash, Pexels, Adobe Stock with proper license, or created/generated by the project owner

Required filenames used by `juego-comida.js`:
- `apple.jpg`
- `banana.jpg`
- `bread.jpg`
- `milk.jpg`
- `water.jpg`
- `rice.jpg`
- `egg.jpg`
- `chicken.jpg`
- `fish.jpg`
- `cheese.jpg`
- `coffee.jpg`
- `tea.jpg`

If an image is missing, the game shows a clean text placeholder with the Spanish and English words.

## Using The Supplied Collage

Save the 4-by-3 collage image as:

`assets/food/source-collage.png`
Then run:

```powershell
& 'C:\Users\user\.cache\codex-runtimes\codex-primary-runtime\dependencies\python\python.exe' tools/prepare_food_images.py assets/food/source-collage.png
```

The script crops the collage left-to-right, top-to-bottom and writes the 12 exact `.jpg` files above.
