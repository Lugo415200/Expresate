-ing

Conecta lo que ya reconoces en español con acciones en inglés.

1
-ando / -endo = -ing Una pista rapida para acciones
Español-ando / -endo
→
Inglés-ing

Cuando ves una acción como corriendo, piensa en una acción con -ing: running.

2
corriendo → running Mira el final de cada palabra
Español corriendo
Inglés running
3
bailando → dancing -ando tambien apunta a -ing
Español bailando
Inglés dancing
4
comiendo → eating La idea es la accion, no letra por letra
Español comiendo
Inglés eating

Cuidado: no siempre suenan parecido. Usa el patron como pista.

5
Une las acciones Español → Inglés

Haz tres parejas correctas.

6
Ahora mira la frase completa Esto prepara I am

EspañolYo estoy corriendo.

InglésI am running.

Primero reconoces running. Después aprendes a decir I am running.

7
Idea final Tu pista para acciones

corriendo = running · bailando = dancing · comiendo = eating
